pysacio by emolch

Python module to read and write binary SAC (Seismic Analysis Code) files.

This is a very short module written in pure Python to read and write the binary SAC (Seismic Analysis Code) file format. The file format is desribed in the SAC manual, available at http://www.iris.edu/manuals/sac/manual.html.

Dependencies

numpy

Install

python setup.py install
